# 🔐 Password Strength Checker

A web-based application that analyzes password strength using rule-based and entropy-based techniques.

## 🚀 Features
- ✔ Real-time password validation (✔ / ❌)
- 📊 Strength meter with percentage
- 🧠 Entropy-based strength calculation
- ⏱ Estimated crack time
- ⚠ Dictionary password detection
- 💡 Smart suggestions for improvement
- 💎 Modern UI with glassmorphism design

## 🛠 Technologies Used
- HTML
- CSS
- JavaScript

## 📌 Project Objective
To promote strong password practices and improve user awareness about cybersecurity by analyzing password strength and preventing common attacks like brute force and dictionary attacks.

## 🎯 Outcome
This project helps users understand how secure their passwords are and encourages the creation of strong and complex passwords.
